Ejercicios resueltos de shell script

  • 5 years ago
  • Posted in:shell script, Sistemas Operativos
  • 1
  • Author: jcmoreno Myfpschool
  • Tags: ejercicios, ejercicios resueltos, linux, script, scripting, shell script, solución ejercicios

1. Realiza un script que compruebe si el usuario actual del sistema es manolo, si es así salúdalo, sino te despides de él amigablemente.

2. Realiza un script que compruebe si estamos a 25 de este mes, si es así mostrará el texto “25 de octubre FU FU FU”.

3. En un fichero tengo una palabra clave. Haz un script que muestre si dicha palabra es “myfpschool” o no.

4. Realiza un script que compruebe si la segunda columna de un fichero que contiene solamente una línea contiene la palabra myfpschool.

5. Realiza un script que compruebe si las tres primeras columnas de un fichero (que contiene solamente una línea) son iguales o diferentes.

6. Realiza lo mismo pero teniendo en cuenta que el fichero está comprimido con gzip.

7. Tengo un tar con tres ficheros. Se pide realizar un script que genere un fichero toatoa.gz con las líneas de los tres primeros ficheros que contengan la cadena “te nesecito”.

8. Tengo un fichero personahes.gz que contiene dos columnas <país de procedencia>. Se desea saber el nombre de los ciudadanos de “chiquitistan”. Ejemplo de fichero:
Juan holanda
Pedro chiquitistan
Ramiro rusia
Josue dinamarca

9. Se pide realizar un script que muestre las líneas de un fichero cualquiera que contengan las palabras uno y dos sin importar sin van en mayúsculas y minúsculas.

10. Idem que el fichero anterior pero que contengan la palabra “a” al comienzo de las frases.
